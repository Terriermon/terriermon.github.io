主要关键:

  1. Xcode工程的Bundle Id
  2. Apple开发中中心配置的App Id
  3. 推送证书对应的App Id

这三个Id必须一致.

下面给出操作步骤截图:

No1. 进入Applde开发中中心，选择Xcode工程配置的Bundle Id一致的App Id.

No2. 启用选中App Id证书中的Push Notification.
